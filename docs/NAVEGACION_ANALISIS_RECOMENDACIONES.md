# Análisis de Navegación y Recomendaciones Técnicas

## 1. Estado actual
- Sidebar desktop: 38 ítems en 8 grupos.
- Bottom nav mobile: 5 ítems + menú expandido con redundancia.

## 2. Modificaciones estructurales
- Transformar Planificación y Ejecución en dashboards de dominio con tabs en vez de listas planas.
- Reducir ítems visibles del sidebar a 12-14.
- Reordenar por flujo de obra real: Dashboard → Proyectos → Seguimiento → Presupuestos → Bodega → Finanzas → Sistema.

## 3. Consolidación de funciones
- RRHH + planilla-destajos → Talento / Mano de obra.
- Calidad + Riesgos + Cumplimiento → Calidad y Cumplimiento.
- Seguimiento + Avances + EVM → Seguimiento Obra.
- Predictivo + Profitabilidad + Exportación → Inteligencia.
- Notificaciones mantener como badge en header; eliminar módulo standalone.

## 4. Optimizaciones de interfaz
- Sidebar desktop: máximo 3 niveles visuales; mobile: 2 niveles.
- Progresivos disclosure: hover expand en collapsed; sheet inferior en mobile.
- Accesos rápidos contextuales en Dashboard para reducir dependencia del menú.
- Acciones globales como exportación/ajustes se mueven al header.
