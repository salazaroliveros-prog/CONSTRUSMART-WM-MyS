# CATEGORÍAS DE RENGLONES POR TIPOLOGÍA CONSTRUCTIVA

## Documento de Referencia para Configuración de Base de Datos

Este documento describe las categorías típicas de renglones constructivos para cada tipología, las cuales pueden usarse para:
1. Clasificar renglones en la base de precios
2. Configurar reglas de aplicación específicas
3. Validar consistencia de presupuestos por tipología

---

## 1. TIPOLOGÍA RESIDENCIAL

### Categorías Principales:
- **Cimentación**: 
  - Cimientos corridos
  - Cimientos aislados
  - Losas de cimentación
  - Impermeabilización de cimientos

- **Estructura**:
  - Columnas y vigas de concreto
  - Losas entrepiso
  - Muros de carga
  - Escaleras

- **Mampostería**:
  - Muros de bloque
  - Muros de ladrillo
  - Divisiones interiores
  - Aplanados

- **Acabados**:
  - Pisos (cerámica, porcelanato, mármol)
  - Paredes (pintura, estuco, papel tapiz)
  - Plafones (yeso, PVC, fibrocemento)
  - Carpintería interiores
  - Herrería interior
  - Vidriería
  - Baños (sanitarios, grifería, accesorios)
  - Cocinas (muebles, encimeras, artefactos)

- **Instalaciones**:
  - Instalación eléctrica (tomacorrientes, interruptores, luminarias)
  - Instalación hidráulica (tuberías, grifos, sanitarios)
  - Instalación sanitaria (desagües, ventilación)
  - Instalación de gas
  - Instalación de aire acondicionado

- **Acabados exteriores**:
  - Fachadas
  - Techos (teja, lámina, impermeabilización)
  - Jardines y áreas exteriores
  - Cercas y portones

---

## 2. TIPOLOGÍA COMERCIAL

### Categorías Principales:
- **Cimentación**:
  - Cimientos reforzados
  - Losas de cimentación de gran capacidad
  - Pilotes y pilas
  - Drenaje de cimientos

- **Estructura**:
  - Estructuras de concreto armado
  - Estructuras metálicas
  - Losas de entrepiso de gran carga
  - Naves industriales
  - Elevadores y escaleras mecánicas

- **Mampostería**:
  - Muros divisorios prefabricados
  - Muros cortafuego
  - Aplanados decorativos

- **Acabados Interiores**:
  - Pisos comerciales (porcelanato, mármol, vinil, epóxico)
  - Paredes corporativas (pintura, madera, vidrio)
  - Plafones suspendidos (acústicos, metálicos)
  - Carpintería comercial (showers, vitrinas, oficinas)
  - Herrería decorativa
  - Vidriería templada

- **Instalaciones Comerciales**:
  - Sistemas eléctricos de alta potencia
  - Iluminación comercial (LED, track lighting)
  - Sistemas de HVAC centralizados
  - Instalación contra incendios (sprinklers, detectores)
  - Sistemas de seguridad (CCTV, acceso)
  - Sistemas de comunicaciones (datos, telefonía)
  - Sistemas hidráulicos especiales

- **Acabados Exteriores**:
  - Fachadas comerciales (vidrio, ACM, piedra)
  - Cubiertas metálicas
  - Estacionamientos
  - Señalización

- **Equipamiento Especial**:
  - Montacargas
  - Puertas automáticas
  - Escaleras mecánicas y ascensores
  - Sistemas de carga

## 3. TIPOLOGÍA INDUSTRIAL

### Categorías Principales:
- **Cimentación Pesada**:
  - Cimientos para maquinaria pesada

  - Pilotes y cajones

- **Estructura Industrial**:
  - Estructuras metálicas pesadas

  - Estructuras de concreto de alta resistencia

- **Instalaciones Industriales**:
  - Sistemas eléctricos de alta tensión
  - Subestaciones eléctricas
  - Sistemas de aire comprimido
  - Sistemas de fluidos industriales
  - Sistemas de ventilación y extracción
  - Sistemas de tratamientos de agua

- **Equipamiento Industrial**:
  - Montacargas industriales
  - Grúas puente
  - Cintas transportadoras
  - Tanques y silos

- **Pisos Industriales**:
  - Pisos de concreto reforzado
  - Pisos epóxicos
  - Pisos antideslizantes
  - Pisos de resistencia química

- **Aislamiento y Recubrimientos**:
  - Aislamiento térmico
  - Aislamiento acústico
  - Recubrimientos resistentes a químicos
  - Recubrimientos antiestáticos

- **Seguridad Industrial**:
  - Sistemas contra incendios industriales

  - Vías de escape
  - Sistemas de contención

## TIPOS DE ACTIVIDAD PARA ESTACIONALIDAD

Las actividades para ajustes estacionales son:
1. **cimentacion** - Trabajos de fundación
2. **estructura** - Estructuras y elementos portantes
3. **mamposteria** - Muros y divisiones
4. **acabados** - Revestimientos interiores y exteriores
5. **instalaciones** - Eléctricas, hidráulicas, mecánicas
6. **movimiento_tierra** - Excavación, relleno, nivelación
7. **pavimentacion** - Vías, estacionamientos, superficies
8. **general** - Trabajos no clasificados

## IMPLEMENTACIÓN EN BASE DE DATOS

Si se desea crear una tabla de categorías de renglones para validación:

```sql
CREATE TABLE IF NOT EXISTS erp_categorias_renglones (
  id uuid DEFAULT uuid_generate_v4() PRIMARY KEY,
  categoria text NOT NULL,
  tipologia text NOT NULL CHECK (tipologia IN ('residencial', 'comercial', 'industrial', 'civil', 'publica')),
  subcategoria text,
  descripcion text,
  activo boolean DEFAULT true,
  created_at timestamptz DEFAULT now() NOT NULL
);

CREATE INDEX idx_categorias_tipologia ON erp_categorias_renglones(tipologia);
CREATE INDEX idx_categorias_activas ON erp_categorias_renglones(activo);
```

Esta tabla permitiría validar que los renglones de un presupuesto sean consistentes con la tipología del proyecto.
